# Underwater-Images-Augmentation
A lightweight Python library providing advanced underwater image augmentation techniques for computer vision and deep learning applications.

Designed specifically to simulate real-world underwater conditions such as turbidity, light scattering, caustics, haze, bubbles, and color attenuation.

## Installation
You can now install the library on your own computer using:
```bash
pip install git+https://github.com/ayan-shahab766/Underwater-Images-Augmentation.git
```

## Features
### Geometric Transformations

- Rotate_Image
- Flip_Horizontal
- Flip_Vertical
- Random_Crop
- Resize_Image
- Mosaic_Augmentation
- Mixup_Augmentation

### Underwater Distortion Effects

- Ripple_Distortion
- Wave_Distortion
- Scattering_Blur
- Channel_wise_Multi_scale

### Environmental Simulation

- Bubbles_Overlay
- Debris_Overlay
- Caustic_Overlay
- Add_Haze
- Add_Fog
- Underwater_Turbidity

### Color & Lighting Adjustments

- Color_Jitter_Brightness
- Color_Jitter_Contrast
- Color_Jitter_Saturation
- Color_Jitter_Hue
- Attenuate_Channel
- Adjust_Gamma
- Multi_Color_Space_Fusion

## Use Cases

- Underwater object detection
- Marine robotics vision systems
- Data augmentation for CNN training
- Robustness testing under environmental distortions
